## Setup / Usage : 

Clone Repo, then

```
$ npm install
```
```
$ npm start

or Run with nodemon

$ npm run dev
```

### MongoDB
Open "config/keys.js" and add your MongoDB URI, local or Atlas

### Go to address : 

http://localhost:5000
